Converts a 3D point represented in the Cartesian coordinate system \((x,y,z)\) to a representation in the Cylindrical coordiante system \((r_C,\theta_C,z_C)\).

\[ r_C = \sqrt{x^2+y^2} \]

\[ \theta_C = atan2(y,x) \]

\[ z_C = z \]

.

See http://mathworld.wolfram.com/CylindricalCoordinates.html and https://en.wikipedia.org/wiki/Cylindrical_coordinate_system

Converts a 3D point represented in the Cartesian coordinate system \((x,y,z)\) to a representation in the Spherical coordiante system \((r_S,\theta_S,\phi_S)\). Note that \(\theta_S\) lays in the \(x\ y\) plane.

\[ r_S = \sqrt{x^2+y^2+z^2} \]

\[ \theta_S = atan2(y,x) \]

\[ \phi_S = atan2(\sqrt{x^2+y^2},z) \]

.

See http://mathworld.wolfram.com/SphericalCoordinates.html and https://en.wikipedia.org/wiki/Spherical_coordinate_system

Parameters
x\( x\ (m)\) The \(x\) cartesian coordinate in meters.
y\( y\ (m)\) The \(y\) cartesian coordinate in meters.
z\( z\ (m)\) The \(z\) cartesian coordinate in meters.
sphericalR\( r_S\ (m)\) The \(r_S\) spherical coordinate (spherical radius) in meters.
sphericalAzimuthInRadians\( \theta_S\ (rad)\) The \(\theta_S\) spherical coordinate (azimuth angle) in radians. Note that the azimuth angle is in the \(x\ y\) plane.
sphericalZenithInRadians\( \phi_S\ (rad)\) The \(\phi_S\) spherical coordinate (zenith angle) in radians. Note that the zenith angle is measured from the z-axis.
See also
SphericalCoordinateToCartesianCoordinate() for reverse coordinate transform.
